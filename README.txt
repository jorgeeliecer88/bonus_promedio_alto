
Si se trabaja proyecto comprimido:

Descomprimir el zip: bonus_promedio_alto
ejecutar en linea de comandos creacion : node index crear 
ejecutar en linea de comandos promedio estudiante : node index promedio -n "nombre"
ejecutar en linea de comandos promedio superior a 3 : node index promedioSuperior

--------------------------

Si se trabaja con proyecto github:

Descargar proyecto:
https://github.com/jorgeeliecer88/bonus_resultados

Ejecutar dentro del proyecto el comando install para instalar descargar dependencias: 
npm install

ejecutar en linea de comandos creacion : node index crear 
ejecutar en linea de comandos promedio estudiante : node index promedio -n "nombre"
ejecutar en linea de comandos promedio superior a 3 : node index promedioSuperior
